# nomadOffice-Server

## Introduction

"Nomad Office - Server" is an Express REST server developed as part of the Capstone Project in the Full Stack Development specialization in Coursera.

## Server hosting

This server has been hosted on the Bluemix platform using a free trial voucher (https://nomadofficeapp.eu-gb.mybluemix.net). When the trial expires, the server will go down.

## Client project

The web client that uses this server can be found in the following repository: 
https://github.com/salvateleco/nomadOffice-Angular

## Take a look

This web has been hosted in the Firebase Hosting service. The web is running and can be accessed here:
https://nomad-office.firebaseapp.com/index.html
